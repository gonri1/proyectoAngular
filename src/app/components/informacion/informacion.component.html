¿Que tiene este ejercicio?

Este ejercicio tiene;

  1. Componente Productos es leido desde un array en Type Script a traves de una interfaz-Modelo
  2. Componente Cards es leido desde el API a traves del servicio HTTP
  3. Componente formulario recoje informacion del formulario a traves de NgModule y lo gestiona por una interfaz-Modelo
  4. Uso de Pipes tanto en Productos para usar imagenes genericas (visto en webinars) y un otro para cambiar el estado de la tarea a completo o incompleto

¿Que es angular?

Angular es un framework de JavaScript de tipo open source, es decir de código abierto, desarrollado por Google. Se utiliza en la creación y programación de aplicaciones web de una sola página.

Como se explica en el sitio web de Angular, “los componentes son los bloques de construcción que componen una aplicación. Un componente incluye una clase de TypeScript con un decorador, una plantilla HTML y estilos”.

  1. ¿Qué es un decorador en angular? El decorador establece la información específica de Angular.
  2. Un selector de CSS define cómo se usa el componente en una plantilla. Los elementos HTML de su plantilla que coincidan con este selector se convierten en instancias del componente.
  3. Una plantilla HTML que le indica a Angular cómo representar el componente
  4. Un conjunto opcional de estilos CSS que definen la apariencia de los elementos HTML de la plantilla
